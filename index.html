wldash

wayland launcher/dashboard

wldash is a convenient dashboard and launcher, allowing easy access to things like date and time, backlight and volume control.

More modules are added as development continues.

To run: 


	git clone https://github.com/kennylevinsen/wldash
	cd wldash
	cargo build --release
	cp ./target/release/wldash /usr/local/bin/
	wldash

To use launcher functionality with dot-desktop: 


	git clone https://github.com/kennylevinsen/dot-desktop
	cd dot-desktop
	cargo build --release
	cp ./target/release/dot-desktop /usr/local/bin/

	eval "$(dot-desktop "$(dot-desktop | wldash)")"
This pipes dot-desktop's selections into wldash, gives the selection back to dot-desktop, and finally evaluates it in a shell.
